Firebase 設定
完成設定後，所有人的資料即時同步
🔥 設定步驟（約 10 分鐘）
1️⃣
前往 Firebase Console 建立專案
→ console.firebase.google.com
點「建立專案」→ 輸入名稱（如 meeting-os）→ 關閉 GA → 建立
2️⃣
建立 Firestore 資料庫
左側選單 → Firestore Database → 建立資料庫
選「以測試模式啟動」→ 選擇地區（asia-east1）→ 啟用
3️⃣
取得 Web App 設定
專案設定（齒輪）→「你的應用程式」→ 點「</>」Web 圖示 → 註冊應用程式
複製 firebaseConfig 物件裡的內容
4️⃣
貼入 index.html 並重新上傳
打開 index.html（記事本或 VS Code）
找到頂部 const firebaseConfig = { 這段
把 YOUR_API_KEY 等欄位換成你複製的內容
存檔後重新上傳到 GitHub → 等 1 分鐘 → 完成！
